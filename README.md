# Fund Insight - 基金博主分析系统

手动输入博主帖子，AI分析预测观点，智能验证准确率。

## 🎯 核心功能

| 功能 | 说明 |
|------|------|
| **手动输入帖子** | 避开小红书反爬，手动粘贴博主内容 |
| **AI智能分析** | 提取预测观点、目标基金、预测周期 |
| **预测周期支持** | 1周/2周/1月/3月/6月/1年 |
| **智能验证** | 根据预测周期自动计算验证时间 |
| **基金数据自动抓取** | 每日更新基金净值，存储历史数据 |
| **准确率统计** | 按博主统计预测准确率 |

## 📦 安装

```bash
# 1. 进入项目目录
cd fund-insight

# 2. 创建虚拟环境
python -m venv venv
venv\Scripts\activate  # Windows

# 3. 安装依赖
pip install -r requirements.txt

# 4. 配置API Key
copy .env.example .env
# 编辑 .env 填写 LLM_API_KEY
```

## 🚀 使用

```bash
# 启动服务
start.bat

# 或手动启动
python -m uvicorn src.api.main:app --host 0.0.0.0 --port 8002
```

访问 http://localhost:8002

## 📋 使用流程

```
1. 添加博主
   ↓
2. 添加帖子（粘贴博主内容，选择发布日期）
   ↓
3. AI分析（提取预测观点）
   ↓
4. 添加基金（输入基金代码）
   ↓
5. 验证预测（自动对比基金涨跌）
   ↓
6. 查看准确率统计
```

## 🔧 核心设计

### 预测验证逻辑

```
博主发布帖子日期: 2024-01-15
预测周期: 1个月
目标验证日期: 2024-02-15

验证时：
- 获取 2024-01-15 的基金净值作为基准
- 获取 2024-02-15 的基金净值作为终点
- 计算涨跌幅，判断预测是否正确
```

### 智能验证

- **看涨预测**: 实际涨了 → 正确
- **看跌预测**: 实际跌了 → 正确
- **震荡预测**: 涨跌幅 < 1% → 正确
