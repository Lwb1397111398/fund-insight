## 1. 功能说明

### 1.1 功能描述
在验证预测页面添加"合并同类预测"按钮，用于合并相同博主对相同基金做出相同判断且目标日期相同的预测（仅置信度不同），置信度取平均值。

### 1.2 实现方案

#### 合并规则
- **相同博主** (blogger_id)
- **相同基金** (fund_code)
- **相同判断** (prediction_type)
- **相同目标日期** (target_date)

#### 置信度计算
- 取所有重复预测的置信度平均值
- 四舍五入取整

#### 数据处理
- 保留第一条预测记录
- 更新置信度为平均值
- 其他重复预测标记为已删除（软删除）
- 记录删除原因
