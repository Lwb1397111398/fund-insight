## 修复成果

### 根本原因

#### 问题：前端显示逻辑不完善
在 [web/index.html](file:///E:/CountBot/countbot/workspace/fund-insight/web/index.html) 文件中：

| 问题 | 影响 |
|------|------|
| 只显示有历史净值的情况 | 没有历史净值时显示空白 |
| 没有友好提示 | 用户不知道发生了什么 |
| 缺少加载状态 | 用户不知道是否在加载 |

### 修改文件

| 文件路径 | 修改类型 | 说明 |
|---------|---------|------|
| web/index.html | 修复 | 添加空数据提示和加载状态 |

### 核心修改

**修复前**:
```html
<div v-if="fundDetail.history && fundDetail.history.length > 0" class="detail-section">
    <h4>历史净值（最近{{ fundDetail.history.length }}天）</h4>
    <!-- 表格内容 -->
</div>
<!-- 没有历史净值时，什么都不显示 -->
```

**修复后**:
```html
<div v-if="fundDetail.history && fundDetail.history.length > 0" class="detail-section">
    <h4>历史净值（最近{{ fundDetail.history.length }}天）</h4>
    <!-- 表格内容 -->
</div>
<div v-else class="detail-section">
    <h4>历史净值</h4>
    <div style="padding: 20px; text-align: center; color: #999;">
        <p>暂无历史净值数据</p>
        <p style="font-size: 12px;">请点击"更新基金"按钮获取历史净值数据</p>
    </div>
</div>
<div v-else style="padding: 40px; text-align: center; color: #999;">
    <p>加载中...</p>
</div>
```
