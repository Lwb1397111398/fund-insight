## 1. 功能说明

### 1.1 功能描述
在待清理界面显示测试数据，并提供一键清理功能，使用硬删除方式彻底删除测试数据。

### 1.2 实现方案

#### 测试数据识别
使用关键词匹配识别测试数据：
```python
TEST_KEYWORDS = ['测试', 'test', 'TEST', 'Test', 'demo', 'Demo', '示例', '样例', '测试数据']
```

匹配范围：
- 博主名称
- 帖子标题、内容
- 基金名称
- 观点内容

#### 删除策略
**硬删除**（直接从数据库删除）：
- 测试博主 → 删除其所有帖子、预测、观点
- 测试帖子 → 删除其关联的预测
- 测试基金 → 删除其关联的预测和历史净值
- 测试观点 → 直接删除

## 3. 接口说明

### 3.1 API 接口

**功能**: 查找所有测试数据

**返回结果**:
```json
{
  "success": true,
  "data": {
    "bloggers": [...],
```

## 5. 使用说明

### 5.1 查看测试数据
