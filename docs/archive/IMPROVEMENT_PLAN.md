# 抓取方式改进说明

## 🎯 问题反馈

你说抓取的还是评论，应该抓取精选频道等有价值的帖子。

## ✅ 发现的解决方案

我发现天天基金网有两种高质量内容源：

### 1. 天天基金网文章（专业文章）
- 网址：https://fund.eastmoney.com/
- 文章链接：https://fund.eastmoney.com/a/202603063664463532.html
- 特点：编辑发布、专业分析、质量很高
- 标题示例：
  - 《ETF今日收评 | 港股创新药相关ETF涨超4% 有色相关ETF跌超2%》
  - 《落袋为安！超200亿 “跑了”》
  - 《如何明确布局十五五开局之年投资蓝图？多家基金公司一线解读》

### 2. 基金吧（原有功能）
- 网址：https://guba.eastmoney.com/list,{fund_code}.html
- 特点：网友评论、高热度、质量参差不齐

## 🚀 改进方案

### 前端添加选项卡
在爬虫弹窗中添加两个选项卡：
- **文章** - 抓取天天基金网专业文章（推荐）
- **帖子** - 抓取基金吧帖子（原有功能）

### API 新增端点
- `POST /api/crawler/fetch-articles` - 抓取天天基金网文章
- `POST /api/crawler/fetch` - 原有基金吧功能（保留）

### 两种抓取方式对比

| 方式 | 来源 | 质量 | 频率 |
|------|------|------|------|
| **文章** | 天天基金网 | ⭐⭐⭐⭐⭐ 很高 | 每天更新 |
| **帖子** | 基金吧 | ⭐⭐⭐ 中等 | 实时更新 |
